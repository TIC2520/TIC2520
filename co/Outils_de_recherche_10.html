L'internaute face à moteur de recherche

Le web est constitué de plusieurs milliards de documents, les internautes peuvent utiliser les moteurs de recherche pour trouver une ressource parmi un ensemble de sites ou de pages internet.

Des questions centrales qui se posent  :

  • Comment fonctionne-t-ils les moteurs de recherche?

  • Recherchent-t-ils les informations en consultant à chaque fois les pages web, selon la demande de l' internaute?

  • L'internaute interroge-ils les pages web au travers le moteur de recherche?

Fondamental : Fonctionnement du moteur de recherche

Des robots, appelés crawlers ou spiders, explorent d'une façon automatique le xeb à des intervalles réguliers, pour que le moteur de recherche puisse localiser en permanence les pages et indexe le contenu dans de gigantesques une bases de données.

L'internaute interroge ces bases de données et non le web complet

Complément : Deep Web

Il existe une part non-négligeable du web totalement inaccessible aux moteurs appelée le Web invisible ou Deep web, qui n'est référencée et accessibles par les moteurs. Il représente un contenu 500 fois plus important que le web classique et de qualité souvent supérieure (https://www.enssib.fr/services-et-ressources/questions-reponses/web-invisible)

Les causes sont multiples:

  • Contenu dynamique ;

  • Contenu non lié ;

  • Contenu à accès limité ;

  • Format non indexable.

Le web invisible représente 60 % de l'ensemble, de pages non indexées (https://www.expressvpn.com/fr/blog/deep-web-vs-dark-web-quelles-differences/)
